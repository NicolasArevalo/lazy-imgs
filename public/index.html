Lazy loading 🌀

En este proyecto, también desarrollado en el curso de manejo del DOM de platzi, se puede evidenciar como se cargan los componentes de forma lazy. En la consola se mostrará información relevante.

Usé la API de Unsplash. Cada vez que presiones un botón, se agregará una imagen de esa categoría. La imagen será la primera que aparezca en la API. Pero la misma, a menos que recargues la página.
